# Design System - Yet Another Store for Brew

This document outlines the visual identity and styling guidelines for **Yet Another Store for Brew**.

## Color Palette

The design is built around a modern, premium Dark Theme featuring a deep store base color combined with a signature Yellow accent.

### Color Tokens

- **Store Base Color (Dark Gray)**:
  - Hex: `#212826`
  - HSL: `hsl(163, 10%, 14%)`
- **Accent Color (Yellow)**:
  - Hex: `#FFC107`
  - HSL: `hsl(45, 100%, 50%)`
- **Subtle Highlight**:
  - Hex: `#FFC1071A` (10% opacity)
- **Text Primary**:
  - Hex: `#F8F8F2`
- **Text Secondary**:
  - Hex: `#ACADAD`

## Visual Assets & Logo Concept

- **Base Structure**: Dark gray store icon outline or 3D block representing the storefront.
- **Accent Elements**: Highlights, borders, or light beams in **Yellow**.
- **Inside the Store**: Stacked small package cubes representing the packages managed by Homebrew.
- **App Tag**: BREW

## Typography

- **Headings**: Outfit or Inter (sans-serif)
- **Body Text**: Inter (sans-serif)
- **Monospace (for CLI output/commands)**: Fira Code or JetBrains Mono
